# Go Utils


## Requirements

- Linux 20.04
- Golang

## Installation

1. `sudo apt install golang-go`

## Run

1. `go build hello.go`

2. `./hello`

## Notes:

1. To create variable: `x int`

2. To assign result of function variable: `x := function()`
